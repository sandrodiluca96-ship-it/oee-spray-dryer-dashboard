# OEE Spray Dryer Dashboard - Release 5.6

Aggiornamento interfaccia e nuova sezione **Production vs Target**.

## Modifiche principali
- Aggiunta sezione laterale **Production vs Target**.
- Lettura produzione su due basi:
  - Physical Powder (kg)
  - Standard Equivalent Output 40% (kg eq.)
- Vista Daily / Weekly / Monthly / Year-to-Date.
- Predisposizione caricamento storico mensile tramite CSV.
- Calcolo baseline iniziale come media mensile storica, quando lo storico viene caricato.
- Nessuna modifica alle formule OEE esistenti.
